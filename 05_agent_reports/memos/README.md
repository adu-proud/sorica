# Analytic Memos

This folder stores the Realm's internal analytic memory.

Mailbox notes are outward-facing messages to the researcher. Analytic memos are inward-facing working notes that help agents track emerging concepts, comparisons, contradictions, and category development.

## Memo Types
- `theoretical`: emerging interpretation or provisional analytic relation
- `comparative`: comparison between fragments, concepts, categories, or source batches
- `contradiction`: negative case, counter-pattern, or adversarial finding
- `memo_on_memos`: Varro synthesis of several memos into a higher-level category

## Rules
- Memos are not final claims.
- Memos must link to source fragments, concepts, or categories using Obsidian links where possible.
- Claims inside memos still require evidence type and evidence level labels when reported outward.
