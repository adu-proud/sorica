# Back-search Protocol

Use before finalizing factual claims.

## Steps
1. Find the source path from the fragment, concept index, source map, or log.
2. Read the surrounding source context.
3. Check source metadata: author, date, participant, institution, source type.
4. Look for adjacent contradiction or qualification.
5. Report source path, confidence, evidence type, and evidence level.

## L2 Rule
L2 material cannot be reported without at least partial source verification.

## Status Values
- `pending`: not checked
- `partial`: source located but context incomplete
- `verified`: source and context checked
