# Serendipity Protocol

Use for L2 clues that are adjacent, anomalous, metaphorical, weak, or oppositional.

## Types
| Type | Meaning |
|---|---|
| `anomaly` | does not fit the current pattern |
| `metaphor` | figurative or analogical clue |
| `contradiction` | challenges an emerging concept |
| `weak_signal` | low-confidence repeated clue |
| `adjacent_case` | nearby but not directly answering |
| `negative_space` | relevant absence or unmapped area |

## Method
1. Name the clue.
2. Type it.
3. State relation to query: `direct`, `adjacent`, `oppositional`, or `speculative`.
4. Back-search at least partially.
5. Send to Mailbox only if the researcher should see it.
6. Write a memo only if it should persist internally.
